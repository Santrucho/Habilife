# Habilife

Aplicacion realizada en Kotlin y Jetpack Compose para Android.

Seguimiento de habitos y objetivos, realizada con las mejores practicas y ultimas tecnologias recomendadas por Google

Tecnologias utilizadas:

-Arquitectura MVVM
-Jetpack Compose
-Navigation
-Material Design
-Firebase Analytics
-Firebase Auth
-Firebase firestore
-LiveData
-ViewModel
-Coroutines
-Dagger Hilt
